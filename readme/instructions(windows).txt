------------------
Installing ChromeDriver
------------------
1. Check Chrome Version in Chrome settings
2. Download Chrome Driver of your Chrome version from 
	https://chromedriver.chromium.org/downloads
3. Select chromedriver_win32.zip
4. Unzip folder
5. Drag chromedriver to Home Directory: /Users/xxxxxxx 




------------------
Running the script
------------------
1. Open Spyder
2. Drag and Drop main.py into the window
3. In the kernal, type: 
	pip install -r requirements.txt
	(Only do this if you have not installed requirements previously)
4. Press 'play' in the top menu
